  - dependencias y entidades de la administración publica federal
      5. 08- Valor de prueba


* La selección de alguna opción de tratamiento institucional determina que perfil se le asigna por defaul al cliente o que bloques se mostraran cuando consteste el cuestionario. **Ver IMAGEN 3**

* Despues contestar el cuestionario se asignar el perfil de acuerdo a la matriz, entonces se debe habilita una opción para imprimir el cuestionario.

* Si vuelve a seleccionar el cuestionario del prospecto se muestra 2 botónes: 
       1. Quiero un perfil con un  MENOR RIESGO
       2. Quiero un perfil con un MAYOR RIESGO
       
* Si selecciona MENOR RIESGO, se muestran opcines de menor riesgo al asignado para poder seleccionar. Al seleccionar guardar se muestra la carta que debe imprimir para que firme el cliente aceptando el cambio. El documento se debe imprimir para anexar a la papeleria y el cuestionario para entregar a contratos.

* Si selecciona MAYOR RIESGO, se le muestra nuevamente el cuestionario para contestar, se imprime para que lo firme el cliente.




REGLAS DE NEGOCIO
* RN001 Para contratos “Personas Físicas Mancomunadas y solidarios” el perfilamiento de prospecto (cuestionario openfinace) es uno para todos los titulares. 

* RN002 - Para personas fisicas, si selecciona **cliente Sofisticado** unicamente contesta 4 preguntas del cuestionario. Adicional debe firmar el anexo 1

* RN003- Para personas físicas, si seleccionar **No sofifisticado** debe contestar todo el cuestionario.

* RN004 - Para personas morales, si selecciona Solicitar  **tratamiento institucional** , se le asigana el perfil 99.

* RN005 Para personas morales, **no** selecciona Solicitar **tratamiento institucional**, debe firmar el anexo 2.

* RN006  Si el cliente  no se desea contestar el cuestionario por **defecto** se asigna el **perfil 1**.

* RN007 De acuerdo al perfil asignado se genera una matriz de Razonabilidad donde se indica a que tipos de inversiones puede operar el cliente.

* RN008 De acuerdo el perfil asignado al cliente se le asigna una matriz de lineamientos de diversificación.

* RN009 Todos los clientes deben tener un perfil de inversión.
